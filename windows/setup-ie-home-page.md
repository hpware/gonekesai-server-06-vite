## 步驟
1. 打開 <kbd>Windows</kbd> 並打入 群組原則管理

2. 點開主機，並按下群組原則物件

3. 創立新的物件叫IE，並用右鍵到編輯

4. 點選系統管理範本，再到 Windows 元件，在這裡我們可以在這裡面做搜尋，因為在這裡慢慢找會很費時間與盡力，點選篩選選項

5. 在篩選關鍵字輸入「首頁」，輸入後就選 Windows 元件
